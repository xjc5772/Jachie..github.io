使用

  • Express 为 Response 相应对象提供了一个方法：render
  • render中第一个参数不能写路径，默认会去项目中的 views 目录查找该模板文件，也就是说 Express 有一个约定：开发人员把所有的视图文件都放到 views 目录中
  • 如果想要修改默认的 views 目录，则可以app.set('views', render函数的默认路径)
