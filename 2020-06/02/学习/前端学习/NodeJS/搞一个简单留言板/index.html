总结

  • Node 不适合从来没有接触过服务端的人学习
  • 如果想要真正的学号服务端，还是老牌的 Java、PHP 这些平台
  • Node 不是特别适合入门服务端，但不代表 Node 不强大，感觉 Node 很厉害，具有经验的人可以玩儿的非常的牛
  • 为什么不大适合新手呢，不适合新手的原因就在于比较偏底层、而且太灵活，很多东西都是底层的东西其实用上框架后也不赖
  • Java、PHP 好入门的原因就在于：这些平台屏蔽了一些底层
